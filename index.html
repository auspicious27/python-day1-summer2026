Module 2: Python Basics and Data Types

Goal: Learn variables, naming rules, data types, type casting, input and output handling.

2.1 Variables in Python

Variable ek labelled box jaisa hota hai jisme value store hoti hai. Agar student ka naam store karna hai, toh student_name variable bana sakte hain. Agar age store karni hai, toh age variable use kar sakte hain.

Variable ko samjhane ka easiest tarika hai box example. Socho aapke paas ek box hai jiske upar label laga hai student_name. Us box ke andar value Aman rakhi hai. Jab bhi program mein student_name likhenge, Python us box ke andar ki value use karega.

Real-world example School register mein columns hote hain: Name, Roll Number, Marks. Python mein variables bhi waise hi data ko naam dete hain. Naam meaningful hoga toh code padhne mein easy hoga.

Logical operators real life rules jaisa kaam karte hain. Example: exam hall mein entry ke liye student ke paas admit card bhi hona chahiye and ID card bhi hona chahiye. Yeh and condition hai. Discount ke liye student ID ho ya coupon code ho, dono mein se ek chalega. Yeh or condition hai.

Mini Project: Student Marks Calculator

Goal: Combine variables, input, type casting, operators, and output in one practical program.

Project Story

Socho teacher ko student ke 3 subjects ke marks enter karne hain aur total, percentage, and pass/fail result nikalna hai. Manually calculator use karne ke bajay Python se ek helper program banayenge. Yeh project beginners ko real programming flow samjhata hai: input lo, process karo, output show karo.

Is project ko class mein ek real school scenario ki tarah present karo. Teacher ke paas student ka naam hai, Math marks hain, Science marks hain, English marks hain. Program pehle data collect karega, phir calculation karega, phir final result readable format mein show karega. Isse learners ko samajh aayega ki programming ka matlab sirf syntax nahi, balki real problem solve karna hai.

Project Code

Run in Jupyter Notebook Cell 
name = input("Enter student name: ")

science = float(input("Enter Science marks: "))

total = math + science + english
percentage = total / 3
is_passed = percentage >= 40

print("Student Name:", name)
print("Total Marks:", total)
print("Percentage:", percentage)
print("Passed:", is_passed)
