Module 1: Introduction To Python

Module 1 mein hum Python ka basic introduction samjhenge. Python kya hai, kyu popular hai, real world mein kaha use hota hai, kaise install karte hain, aur Python script/interpreter kaise run karte hain. Yeh foundation class hai, isliye yahan learners ko programming ka dar hatana important hai.

1. What Is Python?

Python ek high-level programming language hai. High-level ka matlab hai iski syntax human language ke close hoti hai, isliye beginners ke liye easy hoti hai. Python mein code readable hota hai, jaise English instructions. Example: print("Hello") clearly batata hai ki screen par Hello print karna hai.

Story example: Socho aap ek assistant ko bol rahe ho, "Mera naam print karo", "Do numbers add karo", "File read karo". Python wahi assistant hai jo aapki instructions ko computer ke liye executable form mein run karta hai.

print("Welcome to programming") Yeh second line screen par ek aur message print karegi. Python line by line execute karta hai, pehle first print, phir second print.

Python code run karne ke do common ways hain: interpreter and script. Interpreter quick testing ke liye hota hai. Script file actual program ke liye hoti hai. File ka extension .py hota hai, jaise main.py.

Module 2: Python Basics & Data Types

Module 2 mein hum variables, naming rules, data types, type casting, and input/output handling samjhenge. Yeh Python programming ka daily-use foundation hai. Jab bhi program user ka data store karega, calculation karega, input lega, ya output show karega, yeh concepts use honge.

1. Variables & Naming Rules

Variable ek box jaisa hota hai jisme value store hoti hai. Example: student ka naam store karna hai toh name variable bana sakte hain. Marks store karne hain toh marks variable bana sakte hain. Variable names meaningful hone chahiye, taaki code padhkar samajh aaye ki value kis cheez ki hai.

Naming Rules

  • Variable name letter ya underscore se start ho sakta hai.
  • Number se start nahi ho sakta.
  • Spaces allowed nahi hain.
  • Python keywords use nahi karne chahiye, jaise if, for, while.
  • Names case-sensitive hote hain: Name and name different hain.

Practice Code: Variables

student_name = "Aman"
age = 21
marks = 88.5

print(student_name)
print(age)
print(marks)
Line / Code Detailed Explanation
student_name = "Aman" student_name variable mein text value store hui. Quotes ke andar value string hoti hai.
age = 21 age variable mein integer number store hua. Quotes nahi hain, isliye calculation possible hai.
marks = 88.5 marks variable mein decimal number store hua. Decimal number ko float bolte hain.
print(...) Variable ki value screen par show karta hai.

2. Data Types: int, float, str, bool

Data type batata hai ki value kis nature ki hai. Python automatically data type identify karta hai. Agar value number hai toh int/float, agar text hai toh str, agar true/false hai toh bool. Data type samajhna zaroori hai kyunki har type ka behavior alag hota hai.

Practice Code: Data Types

roll_no = 101
percentage = 87.5
name = "Riya"
is_passed = True

print(type(roll_no))
print(type(percentage))
print(type(name))
print(type(is_passed))
Line / Code Detailed Explanation
roll_no = 101 Integer value hai, yani whole number. Output type: <class 'int'>.
percentage = 87.5 Decimal number hai, isliye float type hai.
name = "Riya" Quotes ke andar text hai, isliye string type hai.
is_passed = True Boolean value hai. True/False decision making mein use hoti hai.
type(...) Variable ka data type check karta hai.

3. Type Casting

Type casting ka matlab ek data type ko doosre data type mein convert karna. Example: user se input hamesha string form mein aata hai. Agar input ko calculation mein use karna hai, toh usko int() ya float() se number mein convert karna padta hai.

Practice Code: Type Casting

num1 = "10"
num2 = "20"

total = int(num1) + int(num2)

print(total)

Expected Output

30
Line / Code Detailed Explanation
num1 = "10" Yeh string hai, number nahi, kyunki quotes ke andar hai.
int(num1) String "10" ko integer 10 mein convert karta hai.
total = int(num1) + int(num2) Dono string values ko number mein convert karke addition hota hai.

4. Input / Output Handling

Input ka matlab user se data lena. Output ka matlab user ko result dikhana. Python mein input() user se data leta hai, aur print() output show karta hai. Important point: input() hamesha string return karta hai.

Practice Code: User Input

name = input("Enter your name: ")
age = int(input("Enter your age: "))

print("Hello", name)
print("Next year your age will be", age + 1)
Line / Code Detailed Explanation
input("Enter your name: ") User se name input leta hai. Prompt message screen par show hota hai.
int(input("Enter your age: ")) User age text form mein enter karega. int() us text ko number mein convert karega.
print("Hello", name) String and variable ko comma se print karta hai. Python automatically space add karta hai.
age + 1 Age number hai, isliye calculation possible hai. Next year age calculate hoti hai.

Module 3: Operators & Expressions

Operators symbols hote hain jo values par operation perform karte hain. Example: + addition karta hai, > comparison karta hai, and logical conditions combine karta hai. Expression ka matlab values, variables, and operators ka combination hota hai jo final result produce karta hai.

1. Arithmetic Operators

Arithmetic operators maths operations ke liye use hote hain: addition, subtraction, multiplication, division, remainder, power, and floor division.

Practice Code

a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)

print(a // b)

2. Relational Operators

Relational operators comparison ke liye use hote hain. Inka output hamesha boolean hota hai: True ya False. Example: marks passing marks se zyada hain ya nahi.

Practice Code

marks = 75
passing_marks = 40

print(marks > passing_marks)
print(marks == passing_marks)
print(marks != passing_marks)
Line / Code Detailed Explanation
marks > passing_marks Check karta hai marks passing marks se zyada hain ya nahi. Output: True.
marks == passing_marks Equal comparison karta hai. Assignment = se different hai.
marks != passing_marks Not equal check karta hai. Agar values different hain toh True.

3. Logical Operators

Logical operators multiple conditions combine karte hain. and tab True hota hai jab dono conditions True hon. or tab True hota hai jab at least ek condition True ho. not result ko reverse karta hai.

Practice Code

age = 20
has_id = True

print(age >= 18 and has_id == True)
print(age >= 18 or has_id == True)
print(not has_id)
Line / Code Detailed Explanation
age >= 18 and has_id == True Dono conditions True honi chahiye: age adult ho and ID available ho.
or At least ek condition True ho toh output True hota hai.
not has_id Boolean ko reverse karta hai. Agar has_id True hai, toh not has_id False hoga.

4. Assignment Operators

Assignment operators variable ki value update karne ke liye use hote hain. = value assign karta hai. +=, -=, *= shortcut update operators hain.

Practice Code

score = 10

score += 5
print(score)

score *= 2
print(score)
Line / Code Detailed Explanation
score = 10 Initial value 10 assign hui.
score += 5 Same as score = score + 5. New score 15.
score *= 2 Same as score = score * 2. New score 30.

5. Operator Precedence

Operator precedence ka matlab hai Python pehle kaunsa operation solve karega. Maths ki tarah multiplication/division addition/subtraction se pehle solve hota hai. Brackets sabse high priority rakhte hain.

Practice Code

result1 = 10 + 5 * 2
result2 = (10 + 5) * 2

print(result1)
print(result2)

Expected Output

20
30
Line / Code Detailed Explanation
10 + 5 * 2 Multiplication pehle hota hai: 5 * 2 = 10, then 10 + 10 = 20.
(10 + 5) * 2 Bracket pehle solve hota hai: 10 + 5 = 15, then 15 * 2 = 30.

Mini Practice Project: Student Marks Calculator

Ab Module 1-3 concepts ko combine karke ek small project banate hain. Yeh project variables, input, type casting, arithmetic operators, relational operators, logical operators, and output handling cover karega.

Project Code

name = input("Enter student name: ")

science = float(input("Enter Science marks: "))
english = float(input("Enter English marks: "))

total = math + science + english

is_passed = percentage >= 40

print("Student Name:", name)
print("Total Marks:", total)
print("Percentage:", percentage)
print("Passed:", is_passed)
Line / Code Detailed Explanation
name = input(...) User se student name leta hai and name variable mein store karta hai.
float(input(...)) User marks input karta hai. Input string hota hai, float() usko decimal number mein convert karta hai.
total = math + science + english Teen subject marks add karke total calculate hota hai.
percentage = total / 3 Total marks ko 3 subjects se divide karke average percentage calculate hoti hai.
is_passed = percentage >= 40 Relational operator use hota hai. Agar percentage 40 ya zyada hai toh True, warna False.
print(...) Final output screen par show karta hai.
